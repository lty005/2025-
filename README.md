# top
- 리눅스에서 cpu, 메모리 사용량 확인할때 시용
- 현재 실행 중인 프로세스 목록 표시
- 특정 프로세스 종료 가능


# ps 
- 현재 실행 중인 프로세스 상태를 보여주는 명령어
-  top 실시간으로 보여주지만 ps 당시 상태만 보여준다.
- ps -e 모든 프로세스를 출력
- ps -ef 전체 프로세스를 풀 포맷으로 출력


# jobs
- 현재 터미널에서 실행 중인 작업 목록을 보여줌
- 오직 현재 터미널에서 시작한 작업만 확인 가능

# kill
- 특정 프로세스 종료시키는 명령어
- -9 강제 종료
- -15 정상 종료
